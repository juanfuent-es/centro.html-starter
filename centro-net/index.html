Objetivo

El propósito de este ejercicio es diseñar una página web que reúna y muestre todos tus enlaces sociales en un solo lugar.

Si no utilizas redes sociales o prefieres mantener tu privacidad, puedes listar enlaces a páginas de tu artista favorito, empresa, servicio o cualquier otro recurso de interés. Considera incluir:

  • Al menos tres enlaces a redes sociales (Instagram, X, LinkedIn, etc.).
  • Un enlace de contacto vía correo electrónico.
  • Un enlace de contacto vía WhatsApp.
  • Un enlace de contacto vía teléfono.
  • Una foto de perfil, logo o avatar.
  • Una imagen de background.

Aspectos a evaluar

Estructura y Semántica

  • Usar HTML semántico con etiquetas adecuadas (<header>, <main>, <section>, <footer>, etc.).
  • Utiliza las propiedades obligatorias recomendadas en tags como title en links y alt en imágenes, necesarios para screenreaders.
  • Optimizar la jerarquía de títulos para mejorar la accesibilidad.

Mobile

  • Priorizar la visualización en dispositivos móviles.
  • Utilizar media queries y la etiqueta picture para imágenes adaptativas.
  • Considerar tamaños adecuados para enlaces y texto para facilitar la interacción táctil.

Optimización y Compatibilidad

  • Incluir un favicon compatible con navegadores modernos y fallback para versiones antiguas.
  • Agregar metadata Open Graph para mejorar la previsualización en redes sociales.
  • Utilizar imágenes en formatos modernos (WebP, AVIF) con fallback para compatibilidad.
  • Incluir imágenes responsivas adaptadas a diferentes resoluciones.

Accesibilidad y Diseño

  • Elegir tipografías y tamaños adecuados para mejorar la legibilidad en diversos dispositivos.
  • Asegurar suficiente contraste en los colores utilizados para facilitar la lectura en modo light/oscuro.
  • Proporcionar descripciones alternativas en imágenes mediante el atributo alt.
  • Asegurar que los enlaces sean descriptivos y accesibles (evitar "haz clic aquí").

Tips

Algunas recomendaciones adicionales para mejorar la experiencia:

  • Evita enlaces rotos o que requieran autenticación para ser vistos.
  • Personaliza los enlaces con estilos visuales distintivos.
  • Considera agregar una sección breve de "Sobre mí" para mayor contexto.
